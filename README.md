# scala-web-gradle-spring-jsp-hsql-simple

## Description
A jsp springboot scala gradle build,
that connects to in memory hsql database.

## Tech stack
- springboot
  - jsp
- gradle
  - hsql drivers
  - lombok
  - PostConstruct annotation
- bootstrap
- jquery
- dataTable

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
